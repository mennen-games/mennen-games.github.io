PHPolygon — unsere Engine.

Unsere Spiele laufen auf PHPolygon, einer Game-Engine in PHP, die wir offen auf GitHub entwickeln. Kein Blackbox-Toolkit: du siehst genau, wie sie funktioniert, kannst Fehler melden, mitbauen oder sie für eigene Projekte nutzen. Die Spiele selbst bleiben kommerziell.

Repository öffnen Spiele damit ansehen
game.php
<?php
// Powered by PHPolygon
use PHPolygon\Game;

$game = new Game('Code Tycoon');
$game->scene('main');
$game->run();

Über uns

Aus Idee
wird Spiel.

Mennen Games ist ein unabhängiges Studio ohne festes Genre. Was uns als Idee begeistert, evaluieren wir — und wenn es trägt, bauen wir es. Manches wird ein fertiges Spiel, manches bleibt Experiment.

Unsere Spiele sind kommerziell und tragen das Studio. Die Engine dahinter gehen wir anders an: PHPolygon, eine Game-Engine in PHP, entwickeln wir offen auf GitHub — transparent und zum Mitmachen.
